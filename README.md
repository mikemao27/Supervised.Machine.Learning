## Andrew Ng's Coursera Course Titled "Supervised Machine Learning: Regression and Classification"
This is a work-through of the course "Supervised Machine Learning: Regression and Classification." Within each section are detailed notes pertaining to each of the modules and videos as well as snippets of Jupyter Notebook code that serve as solutions to some of the exercises within the course. 

Please note that this Github Repository is not a comprehensive guide to Andrew Ng's course. It should not be viewed as a substitute for personally enrolling within the course and learning the content oneself. In addition, none of the work provided should be `Copy + Paste` and used as one's own solutions to any of the exercises provided in the official course. Any academic dishonesty is prohibited. The Jupyter Notebook files provided are merely my own solutions to the exercises in the Coursera course and should not be used in any way that violates the academic integrity agreements within Andrew Ng's course.

As per usual, submitting or attempting to submit work for the Coursera course that is not one's own may result in automatically failing the course and/or termination of one's Coursera account (if such information was not already clear).

## Table of Contents
Each folder within this Github Repository can be taken as one section within the course. Within each folder are `.md` Markdown files which contain detailed (but not cumulative) notes on the lectures as well as `.ipynb` Jupyter Notebook files that provide code for the labs and exercises. Neither of these resources should be seen as cumulative.
